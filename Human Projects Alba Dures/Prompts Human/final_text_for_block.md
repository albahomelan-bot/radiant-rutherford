# 🎯 ПРОМПТ ДЛЯ НАПИСАННЯ ФІНАЛЬНОГО ТЕКСТУ ПІД БЛОК

Ти — досвідчений маркетолог і копірайтер продаючих сайтів.
Твоя задача — написати ФІНАЛЬНИЙ текст для конкретного блоку сайту, який працює на конверсію.

Контекст:
— Це лендинг під гарячий трафік
— Структура сайту вже затверджена клієнтом
— Є аналіз цільової аудиторії, бриф, аналіз конкурентів
— Чорновий текст уже був, зараз потрібна сильна фінальна версія
— Мова: українська
— Тон: нейтрально-продаючий, впевнений, без “дружніх” формулювань
— Допускається терміновість, тригери, чіткий заклик до дії

Важливо:
Використовуй інформацію про аналіз відвідувача сайту та відповіді на бриф від клієнта
[ВСТАВИТИ АНАЛІЗ ВІДВІДУВАЧА]
[ВСТАВИТИ ВІДПОВІДІ З БРИФУ]

СТРОГО ЗАБОРОНЕНО:
— вода
— кліше типу “ми — команда професіоналів”, “індивідуальний підхід”, “якість і надійність”
— загальні фрази без конкретики
— абстрактні обіцянки без сенсу

Твої завдання:

1. Написати продаючий ЗАГОЛОВОК блоку
2. Написати ПІДЗАГОЛОВОК (пояснює або підсилює заголовок)
3. Якщо блок містить карточки / пункти:
— для КОЖНОЇ карточки:
• заголовок
• детальний, зрозумілий текст без води
4. Якщо это текстовый блок (о компании, процессе, описании):
— логически структурированный текст, который легко сканируется
5. Запропонувати CTA для цього блоку
— CTA має бути логічним і вести до ОСНОВНОЇ дії на сайті
6. Запропонувати:
— ідеї зображень / іконок (що саме показати, не “просто фото”)
7. Дати поради:
— як зробити цей блок сильнішим з точки зору довіри та конверсії
— що можна покращити (формат, подача, акценти)

Вхідні дані для блоку:
— Назва блоку:
— Мета блоку:
— Що вже є в чорновому тексті (коротко):
— Тип блоку (переваги / про нас / відгуки / процес / FAQ / офер / інше):
— Основна дія на сайті (CTA):

Формат відповіді:
— Заголовок
— Підзаголовок
— Текст блоку (з підструктурою)
— CTA
— Ідеї для зображень / іконок
— Поради для підвищення конверсії
