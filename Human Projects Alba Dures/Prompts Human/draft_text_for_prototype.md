# ✍️ ПРОМПТ ДЛЯ ЧОРНОВОГО ТЕКСТУ ДЛЯ ПРОТОТИПУ

Уяви, що ти досвідчений маркетолог і копірайтер продаючих сайтів
з 20-річним досвідом.

Твоя задача — написати чорновий продаючий текст для сайту,
який буде використовуватись у ПРОТОТИПІ на конструкторі

Це фінальний текст.
Ціль — щоб у прототипі не було пустих місць,
а структура і логіка сайту були зрозумілі клієнту.

Ніша бізнесу:
[ВСТАВ НІШУ]

Тип трафіку:
[гарячий / теплий / змішаний]

Аналіз відвідувача:
[ВСТАВ АНАЛІЗ ВІДВІДУВАЧА — страхи, бажання, проблема, легка дія]

Відповіді на бриф від клієнта:
[ВСТАВ ВІДПОВІДІ НА БРИФ]

Структура сайту по блоках:
[ВСТАВ СПИСОК БЛОКІВ У ПОРЯДКУ]

Вимоги:

- Пиши простою, живою мовою
- Без складних маркетингових слів
- Краще прості формулювання, ніж “ідеально продаючі”
- Если нет данных — выдумывай логично из головы, но реалистично
- Кожен блок має:
• заголовок
• підзаголовок
• детальний текст в карточках
• маркери, если целесообразно
• CTA (если уместно)
- Для кожного блоку запропонуй:
• приклад картинки або тип зображення (що туди підійде)

Формат відповіді (ОБОВʼЯЗКОВО):

Блок 1 — [Назва блоку]
Заголовок:
Підзаголовок:
Основний текст / маркери:
CTA:
Картинка / візуал:

Блок 2 — …

Пиши так, ніби цей текст одразу копіюють у стандартні блоки конструктору
