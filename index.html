AI Social Media Assistant v1.01

Autonomous SMM content-factory in your pocket

Project Overview

An autonomous agent built on n8n automation pipelines and OpenAI model APIs. It converts raw voice dictates or brief texts into ready-to-publish SMM posts with custom graphics and handles multi-channel cross-posting in seconds.

Key Features

  • Voice-to-Text: Uses OpenAI Whisper to transcribe spoken thoughts immediately.
  • Copywriting Agent: GPT-5-mini processes context and generates natural-sounding posts with micro-imperfections for maximum authenticity.
  • Illustrator Agent: Generates flat-style graphical illustrations tailored to the post's context, optimized to under 500 KB.
  • Google Sheets Logging: Automatically logs theme, description, date, and link into a centralized spreadsheet.

Stack & Tools

n8n OpenAI API Zernio API LangChain Google Sheets

15s vs. 1 hour manual posting
