The Problem (Scanner Clutter) Проблема (Незручні сканери)

Scanning documents using mobile phone cameras often results in separate files, distorted angles, or poor contrast. Traditional scanner apps are bloated, require paid subscriptions to remove watermarks, display endless advertisements, or restrict page counts.

Сканування документів на камеру смартфона часто дає викривлені зображення з поганою яскравістю та тінями. Мобільні додатки-сканери вимагають платні підписки, заважають рекламою, обмежують кількість сторінок або додають свої логотипи на документи.

The Solution (Secure Utility Bot) Рішення (Приватний бот-сканер)

An n8n-powered Telegram bot. Send any number of document photos in one message. The workflow downloads them, processes the image binary data to adjust margins, runs text extraction (OCR), merges the pages into one PDF, and returns it.

Телеграм-бот на базі n8n. Надішліть будь-яку кількість фото документів пакетом. Сценарій завантажить зображення, вирівняє їх, розпізнає текст через OCR, склеїть у єдиний файл PDF та поверне готовий результат за кілька секунд.

System Flow Крок за кроком

Image Processing & Compilation Flow Архітектура обробки зображень

1

Bulk Photo Upload Пакетне завантаження фото

The user sends photos of pages as a group. The Telegram node triggers the workflow and waits to collect all binary image streams. Користувач надсилає групу фотографій. Телеграм-тригер в n8n фіксує пакетну передачу та збирає всі бінарні потоки зображень.

2

Binarization & Perspective Crop Вирівнювання та перетворення

Images undergo automated digital processing to correct perspective angles, increase black-and-white text contrast, and crop deskewed margins. Кожне зображення проходить автоматичну обробку для коригування кутів нахилу, посилення чорно-білого контрасту та обрізки полів.

3

OCR Text Injection Впровадження OCR-тексту

An OCR engine processes the pages, extracting selectable text and embedding it as an invisible searchable overlay in the document layer. Вузол OCR зчитує символи на сторінках, створюючи невидимий текстовий шар, що дозволяє виділяти та шукати текст усередині PDF.

4

PDF Merging & Delivery Склеювання PDF та доставка

n8n merges the individual processed pages into a single PDF file and sends it back to the user's chat window as an attachment. n8n збирає окремі оброблені сторінки в один PDF-документ і надсилає його назад у чат користувача файлом.
