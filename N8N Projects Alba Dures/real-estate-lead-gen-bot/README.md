# Автоматическая система лидогенерации и прогрева для агентств недвижимости (WhatsApp + n8n)

Этот проект представляет собой готовое решение для автоматического поиска, квалификации и прогрева собственников недвижимости. 

## 🗺 Локальный бизнес-потенциал (Албания / Дуррес / Тирана)
Рынок недвижимости в Албании (особенно в курортных городах Дуррес, Влёра, Саранда и столице Тиране) переживает бум. 
* **Проблема местных агентств:** Огромный объем коммуникации ведется исключительно вручную через WhatsApp. Агенты не успевают оперативно отвечать арендодателям и клиентам, теряя лиды.
* **Решение:** Продажа этой системы как готового продукта (Saas/Шаблон под ключ) для локальных албанских агентств недвижимости. Стоимость внедрения такого шаблона для одного агентства может составлять **€300 – €500** + ежемесячное обслуживание за поддержку n8n сервера и мониторинг.

## 🛠 Техническое задание проекта (Оригинал)

### 1. Общая задача
Разработка архитектуры системы автоматической лидогенерации для агентства недвижимости с использованием n8n, Google Sheets, WhatsApp Business API и искусственного интеллекта.

### 2. Исходные данные
* Google Sheets (в качестве CRM);
* n8n (как ядро автоматизации);
* OpenAI / Gemini API (как AI Agent);
* WhatsApp Business API (или альтернативные шлюзы).

### 3. Бизнес-процесс
Менеджер вручную вносит нового собственника недвижимости в Google Sheets. 
Таблица содержит поля:
* `ID`, `Телефон`, `Ссылка на объявление`, `Имя`, `Район`, `Цена`, `Статус`, `Этап AI`, `Последнее сообщение`, `Вероятность договора`, `Комментарий`.

Система должна:
1. Автоматически обнаруживать новую строку в таблице.
2. Инициировать переписку в WhatsApp.
3. Вести естественный диалог, используя контекст и память предыдущих сообщений.
4. Автоматически обновлять Google Sheets после каждого сообщения (менять статус, этап переговоров, вероятность заключения договора).
5. При достижении согласия — передавать готового лида живому менеджеру.
