# 🏃‍♂️ MVP Система автоматизації пошуку та публікації спортивних подій (Finishers -> Telegram/Facebook)

Це технічне завдання (ТЗ) збережено як перспективний проект для портфоліо та майбутньої реалізації.

---

## 🎯 Суть проекту:
Створення MVP-системи, яка автоматично шукає нові спортивні події на сайті **Finishers.com**, обробляє їх за допомогою ШІ (переклад, адаптація українською, створення анонсу), надсилає адміністратору в Telegram-бот на погодження, а після схвалення ставить у чергу та автоматично публікує в Telegram-канал та на Facebook-сторінку з релевантним фото та прямим посиланням на сайт організатора.

---

## 🛠️ Запропонована архітектура та стек:
* **Платформа автоматизації:** n8n (self-hosted на VPS).
* **AI-Ядро:** OpenAI API (GPT-4o-mini).
* **База даних / Логи:** Google Sheets.
* **Канали дистрибуції:** Telegram Bot API (модерація та публікація), Meta Graph API (публікація у Facebook).

---

## 📋 Повне ТЗ від замовника:

### 1. Отримання даних (Finishers)
* Регулярна перевірка сайту Finishers на предмет нових подій за заданими критеріями.
* Збір полів: назва, категорія, дата, локація, дистанції, маршрут, ціна, дедлайн реєстрації, офіційне посилання на сайт події/організатора.
* Захист від повторної обробки (дублікації) через базу даних Google Sheets.

### 2. ШІ-обробка (AI Engine)
* Відбір фактів, переклад та адаптація українською.
* Генерація живого анонсу у заданому стилі без вигадування фактів.
* Можливість змінювати системний промпт ШІ без втручання в код.

### 3. Робота з фотографіями (Рішення)
* **Варіант A (Пріоритет):** Парсинг `og:image` з офіційного сайту події (ліцензійно чисте промо-фото).
* **Варіант B (Резерв):** Авто-пошук тематичних фото на безкоштовних стоках (Unsplash/Pexels API).
* **Варіант C (Ручний):** Адміністратор може надіслати власне фото як Reply на повідомлення модерації в Telegram.

### 4. Модерація в Telegram
* Повідомлення-прев'ю з кнопками `✅ Погодити` / `❌ Відхилити` / `✏️ Редагувати`.
* Схвалені пости автоматично потрапляють у чергу публікації.

### 5. Публікація та Черга
* Черга працює за Cron-розкладом (наприклад, публікація кожні 4 години).
* Автопостинг у Telegram-канал та Facebook-сторінку через офіційні API.
