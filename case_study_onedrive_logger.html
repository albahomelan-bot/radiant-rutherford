Telegram to OneDrive Logger

Automated Instant Messaging & Document Sync Автоматичний хмарний реєстратор нотаток

An n8n microservice that links Telegram chat messages directly to Microsoft OneDrive. It downloads current text files from cloud storage, structures and formats new inputs with JS timestamps, and re-uploads updated documents.

Мікросервіс на базі n8n, що зв'язує чат Telegram напряму з хмарою Microsoft OneDrive. Він завантажує текстові файли логів, додає нові записи з часовими мітками за допомогою JS та перезаписує оновлений документ.

The Problem (Scattered Notes) Проблема (Втрачені думки)

Writing down fast ideas, server issues, tasks, or metrics during development usually ends up in chaotic messenger chats, saved messages, or random notepad apps. Accessing cloud files on mobile, waiting for apps to open, and editing text manually is slow and distracting.

Запис швидких думок, завдань чи метрик розробки зазвичай ведеться хаотично: у чатах месенджерів чи випадкових нотатниках. Відкривати хмарні диски на телефоні, чекати завантаження додатку і редагувати списки вручную — довго та незручно.

The Solution (Instant Chat Logger) Рішення (Реєстратор у чаті)

A simple Telegram bot that listens to your messages. Type or paste anything, and the n8n backend instantly pulls your task file from Microsoft OneDrive, runs a clean JS format to prepend a timestamp, appends the new text, and saves it back to the cloud.

Простий Telegram-бот, який слухає повідомлення. Ви пишете будь-яку фразу, а сценарій n8n миттєво підтягує текстовий файл з Microsoft OneDrive, форматує за допомогою JS додаючи часову мітку, дописує новий текст та повертає його у хмару.

System Flow Крок за кроком

OneDrive Sync Process Схема синхронізації з OneDrive

1

Telegram Input Telegram-введення

User sends tasks, logs, or quick notes to a secure Telegram bot channel. Користувач надсилає завдання, лог або коротку нотатку до безпечного Telegram-бота.

2

OneDrive File Fetch Завантаження файлу з OneDrive

n8n authenticates via Microsoft Graph API and downloads the active log file (e.g. `todo_log.txt`) from OneDrive. n8n підключається через Microsoft Graph API та скачує активний файл журналу (наприклад, `todo_log.txt`) з OneDrive.

3

JS Formatting & Append JS-форматування та дозапис

A custom Code Node formats the new line, prepending the local date, time, and text parameters before appending it to the downloaded content. Кастомний JavaScript-код форматує новий рядок, додаючи локальну дату, час та параметри тексту до завантаженого контенту.

4

Re-upload & Response Завантаження оновлень та звіт

n8n uploads the modified file back to OneDrive, overwriting the old version, and sends a checkmark confirmation to the user. n8n перезаписує файл у хмарі OneDrive новою версією та надсилає користувачеві галочку підтвердження у чат.
