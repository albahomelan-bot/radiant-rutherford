# 🇦🇱 Бот-переводчик албанского языка (Albanian Translator Bot)

Этот проект содержит воркфлоу n8n для Telegram-бота, который автоматически переводит входящие текстовые и голосовые сообщения с русского/украинского на албанский язык с помощью модели **Gemini 2.5/3.5 Flash** и **OpenAI Whisper**.

---

## 📂 Структура папки проекта:
1. `albanian_translator_bot.json` — Готовый к импорту JSON-файл сценария n8n.
2. `README.md` — Этот файл с описанием архитектуры и рекомендациями по улучшению.

---

## 🛠️ Описание текущей архитектуры воркфлоу:
1. **Telegram Trigger** — Принимает сообщения от пользователя в реальном времени.
2. **Is Voice Message (IF)** — Проверяет, содержит ли сообщение голосовой файл:
   - **Да:** Скачивает аудио (`Download Voice File`), отправляет в OpenAI Whisper (`Transcribe Voice`), переводит в текстовый формат (`Voice Text`).
   - **Нет:** Сразу считывает текст сообщения (`Message Text`).
3. **Translate to Albanian (LLM Chain)** — Задает системный промпт для перевода, транскрипции кириллицей и добавления трех разговорных фраз.
4. **Gemini Model (Shared)** — Модель ИИ (рекомендуется `models/gemini-3.5-flash`), выполняющая перевод.
5. **Send Translation (Telegram)** — Отправляет готовый перевод пользователю в чат.

## 💡 Рекомендации по улучшению бота:

### 1. 🔊 Автоматическая озвучка перевода (Text-to-Speech)
Чтобы тренировать не только чтение, но и понимание на слух, научим бота отправлять вам голосовое сообщение с правильным албанским произношением.

**Как настроить в n8n:**
1. После ноды **`Translate to Albanian`** добавьте ноду **`HTTP Request`**.
2. **Настройки ноды HTTP Request:**
   - **Method:** `GET`
   - **URL:** `https://translate.google.com/translate_tts?ie=UTF-8&tl=sq&client=tw-ob&q={{ encodeURIComponent($json.text.split('Перевод:')[1].split('\n')[0].trim()) }}`
     *(Этот запрос автоматически берет из ответа ИИ только албанскую строку перевода и генерирует для нее аудиофайл)*.
   - **Response Format:** `File` (выходной файл будет сохранен как бинарные данные).
3. После HTTP Request добавьте ноду **`Telegram`** с операцией **`Send Audio`** или **`Send Voice`**, передав в поле файла полученные бинарные данные `data`.

---

### 2. 🏋️‍♂️ Режим тренажера (Треннинг по карточкам)
Сделайте из бота полноценный тренажер слов. По команде `/test` бот будет присылать вам случайное русское слово и ждать перевод на албанском.

**Как настроить в n8n:**
1. **Создайте Google Таблицу** со столбцами: `Russian`, `Albanian`. Впишите туда слова, которые учите.
2. В Telegram-триггер добавьте проверку: если сообщение равно `/test`, запускается ветка тренировки.
3. Нода **`Google Sheets (Read)`** выбирает случайную строку из таблицы и отправляет русское слово пользователю: *«Переведите на албанский: [слово]»*.
4. Бот сохраняет состояние (какое слово сейчас тестируется) во временную память или базу данных.
5. Когда вы присылаете ответ, ИИ (Gemini) сравнивает ваш ввод с правильным албанским словом из таблицы. Если ответ совпадает (или написан с минимальной опечаткой) — бот поздравляет вас, если нет — показывает правильный вариант.

---

### 3. 🛡️ Повышение отказоустойчивости (Error Handling)
При отправке тихих или поврежденных голосовых файлов Whisper может выдать ошибку, из-за чего весь сценарий n8n остановится.

**Как настроить в n8n:**
1. Откройте настройки ноды **`Transcribe Voice`** (вкладка *Settings*).
2. В параметре **`On Fail`** выберите **`Continue (using error output)`**.
3. Добавьте после нее ноду **`If`**, которая проверяет, успешно ли выполнена транскрибация.
   - Если произошла ошибка — бот присылает сообщение: *«⚠️ Не удалось распознать голосовое сообщение. Пожалуйста, запишите его повторно и громче.»*
   - Если все ок — сценарий идет дальше по стандартной ветке.
